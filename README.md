# 🛠️ HacksTool v1.0
**HacksTool** es una suite de simulación de penetración y recolección de información diseñada para entusiastas de la ciberseguridad.

## ✨ Características
- 🔍 **Domain Info:** Análisis de cabeceras y servidores.
- 📡 **IP Tracker:** Rastreo de registros MX y geolocalización por prefijo.
- 🔓 **Port Scanner:** Detección de puertos abiertos en tiempo real.
- ⚡ **Stress Test:** Simulación visual de ataques de desbordamiento.

## 🚀 Instalación

```bash
git clone [https://github.com/Dev-FelixOfc/HackTool](https://github.com/Dev-FelixOfc/HackTool)
cd HackTool
```

```bash
chmod +x install.sh
./install.sh
```
